# MCP Tools Complete Reference

haz build o parecido antes de terminar un mensaje para asegurar que el codigo no de fallos

## Serena MCP (LSP-based Code Analysis)
- `find_symbol` - Locate specific symbols in codebase
- `find_referencing_symbols` - Discover references and dependencies
- `get_symbol_documentation` - Retrieve symbol documentation
- `insert_after_symbol` - Insert code after a symbol
- `edit_symbol` - Edit existing symbol
- `onboarding` - Initialize and index project

## Context7 MCP (Documentation Retrieval)
- `resolve-library-id` - Resolve library identifier
- `get-library-docs` - Get version-specific documentation
- Usage syntax: "use context7" or "use library /[org]/[project]"

## Sequential Thinking MCP (Structured Reasoning)
- `process_thought` - Process individual thought step
- `generate_summary` - Generate summary of reasoning chain
- Stages: Problem Definition, Research, Analysis, Synthesis, Conclusion

## Magic UI MCP (Frontend Components)
- `get_all_components` - List all available components
- `get_component_by_path` - Get specific component implementation
- `search_components` - Search components by query

## Web Search (Best Practices Research)
- `web_search` - Search current patterns and solutions
- `web_fetch` - Fetch complete documentation pages

Memory Bank MCP Tools

memory_bank_read - Read memory bank files from projects
