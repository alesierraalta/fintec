# 🎨 Paleta Armónica Azul-Verde para FinTec

## Colores Base
- **Azul Principal**: `#10069f` (marca, headers, elementos principales)
- **Azul Secundario**: `#455cff` (acentos, hover states)
- **Verde Éxito**: `#4ade80` (ingresos, logros, elementos positivos)

## Colores de Armonía
- **Teal Intermedio**: `#06b6d4` (transición azul-verde)
- **Verde Azulado**: `#10b981` (emerald-500, para balances positivos)
- **Azul Verdoso**: `#0891b2` (cyan-600, para datos neutrales)

## Uso Psicológico por Contexto

### 🔵 Azules (#10069f, #455cff)
- **Botones principales** y CTAs
- **Headers** y títulos importantes
- **Elementos de marca** y branding
- **Navegación** y estructura

### 🟢 Verde (#4ade80)
- **Ingresos** y transacciones positivas
- **Logros** y metas cumplidas
- **Indicadores de éxito**
- **Crecimiento** de patrimonio

### 🔷 Gradientes Armónicos
- `from-[#10069f] via-[#06b6d4] to-[#4ade80]` - Transición completa
- `from-[#455cff] to-[#10b981]` - Balance azul-verde
- `from-[#10069f] to-[#455cff]` - Gradiente azul puro (brand)

## Aplicación en UI
- **Dashboard**: Azules para estructura, verdes para métricas positivas
- **Botones**: Azul para acciones, verde para confirmaciones/éxito
- **Cards**: Gradientes sutiles azul-teal-verde
- **Estados**: Verde = positivo, Azul = neutro/informativo
